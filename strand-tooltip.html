strand-tooltip

A custom styled tooltip which accepts variable and partially styleable content.

Example HTML Editor

<style> #defaultTarg { color: #438ECA; margin-bottom: 20px; } </style> <!-- Default Example --> <strand-icon id="defaultTarg" type="info" width="14" height="14"></strand-icon> <strand-tooltip target="#defaultTarg"> <label>Hi there, I'm a tooltip.</label> </strand-tooltip> <!-- User dismissable example --> <strand-action id="actionTarg" underline="true"><label>Awesome</label></strand-action> <strand-tooltip tip-width="200" auto="false" target="#actionTarg"> <label><span class="bold">Hi there: </span>you'll have to click the x to dismiss me. Here's a <a href="http://www.example.com" target="_blank">link</a>, too!</label> </strand-tooltip>
