strand-action

A fully-styled inline text link with optional icon.

Example HTML Editor

<!-- Text Example --> <strand-action href="http://www.google.com"> <label>Just a link</label> </strand-action> <!-- Text with Icon Example --> <strand-action href="http://www.google.com"> <strand-icon type="favorite" width="16" height="16"></strand-icon> <label>Awesome</label> </strand-action> <!-- Text, Icon, Underline and Target, Example --> <strand-action href="http://www.github.com" underline target="_blank"> <strand-icon type="favorite" width="16" height="16"></strand-icon> <label>Underline Me</label> </strand-action>
