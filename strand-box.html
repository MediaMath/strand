strand-box

A simple flexbox wrapper for attribute based styling.

Example HTML Editor

<!-- Example Style --> <style> .box { min-width: 65px; min-height: 65px; background: #ff0000; } .box:not(:last-of-type) { margin-right: 5px; } </style> <strand-box direction="horizontal" align="stretch" unresolved> <div class="box"></div> <div class="box"></div> <div class="box"></div> <div class="box"></div> <div class="box"></div> </strand-box>
