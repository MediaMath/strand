strand-spinner

An animated spinning graphic which can be dispalyed during indeterminite loading states.

Example HTML Editor

<strand-spinner radius="9" id="sp1" fill-color="#333333"></strand-spinner> <strand-spinner radius="16" inner-radius="0.8" line-width="0.2" id="sp2" fill-color="#333333"></strand-spinner> <strand-spinner radius="24" inner-radius="5" line-width="10" line-weight="1.5" id="sp3" fill-color="#333333"></strand-spinner> <strand-spinner radius="32" id="sp4" fill-color="#333333"></strand-spinner>

Public Properties

Property Description
ref : Object  
options : none
attribute : ref
reflected to attribute : no 		Returns a referece to the component itself.
radius : Number (optional) 
default : 9
options : none
attribute : radius
reflected to attribute : no 		The radius of the spinner (which is always circular). As expected, radius of 9 yeilds an 18px circle.
innerRadius : Number (optional) 
default : 0.5
options : none
attribute : inner-radius
reflected to attribute : no 		The inner radius of the spinner from where the line tick starts drawing. Values greater than 1 are pixel based - less than 1 is proportional to radius (ie. 0.5 == 50% of radius)
numTicks : Number (optional) 
default : 12
options : none
attribute : num-ticks
reflected to attribute : no 		The number of ticks (lines) which compose the circumference of the spinner. Defaults to 12 for a 'standard' analog clock look.
speed : Number (optional) 
default : 1.33
options : none
attribute : speed
reflected to attribute : no 		Spinner speed in cycles per second.
lineWidth : Number (optional) 
default : 0.5
options : none
attribute : line-width
reflected to attribute : no 		The width of each tick (line) which compose the circumference of the spinner. Values greater than 1 are pixel based - less than 1 is proportional to radius (ie. 0.5 == 50% of radius)
lineWeight : Number (optional) 
default : 0.5
options : none
attribute : line-weight
reflected to attribute : no 		The thickness of each tick (line) which compose the circumference of the spinner. Values greater than 1 are pixel based - less than 1 is proportional to the square root of radius (ie. 0.5 == sqrt(radius) * 0.5)
fillColor : String (optional) 
default : #ffffff
options : none
attribute : fill-color
reflected to attribute : no 		The hex color number to fill each tick (line) which compose the circumference of the spinner. Expects a 7 character string (i.e. '#ffffff').
bgColor : String (optional) 
default : null
options : none
attribute : bg-color
reflected to attribute : no 		The hex color number to fill the background of the spinner. The default is null to leave the background transparent.
paused : String (optional) 
default : false
options : none
attribute : paused
reflected to attribute : yes 		Whether or not the animation is running.
