strand-collection

A simple collection wrapper component useful for 2 way binding to lists of data

Example HTML Editor

<template is="dom-bind"> <strand-collection id="searchResults" data="{{dataList}}" auto="false"> </strand-collection> <template is="dom-repeat" items="{{datalist}}"> <div>{{item.name}}</div> </template> </template>
