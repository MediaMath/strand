mm-pulldown-button

A fully-styled pulldown button, with multiple options.

Example HTML Editor

<mm-pulldown-button type="primary" unresolved> <mm-list-item>single 3PAS</mm-list-item> <mm-list-item>bulk 3PAS</mm-list-item> <mm-list-item>Upload Assets</mm-list-item> <mm-icon type="plus" width="12" height="12" shapeRendering="auto" optimize="true"></mm-icon> <label>Add Creative</label> </mm-pulldown-button> <mm-pulldown-button type="secondary" unresolved> <mm-list-item>single 3PAS</mm-list-item> <mm-list-item>bulk 3PAS</mm-list-item> <mm-list-item>Upload Assets</mm-list-item> <label>Add Creative</label> </mm-pulldown-button>

Example Output

single 3PAS bulk 3PAS Upload Assets single 3PAS bulk 3PAS Upload Assets
