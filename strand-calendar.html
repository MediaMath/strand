strand-calendar

A simple calendar rendering component. Intended for use in strand-datepicker but can be used for standalone date selection. Note: strand-calendar makes heavy use of moment.js

Example HTML Editor

<!-- Single Calendar example --> <template is="dom-bind"> <div> <p>Selected Date: <span>{{selectedDate}}</span></p> <strand-calendar date="{{selectedDate}}"></strand-calendar> </div> </template> <!-- Dual Calendar example --> <template is="dom-bind"> <div> <p>Dual Calendar Example</p> <strand-calendar date="{{leftDate}}" pair-date="{{rightDate}}" disable-past="true"></strand-calendar> <strand-calendar date="{{rightDate}}" pair-date="{{leftDate}}" disable-past="true"></strand-calendar> </div> </template>
