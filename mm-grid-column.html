mm-grid-column

Provides configuration for data fields in mm-grid.

Example HTML Editor

<mm-grid id="mmTestGrid" unresolved style="width: 100%; height: 400px;"> <mm-grid-column field="first_name" width="20%" resize sort>First Name</mm-grid-column> <mm-grid-column field="last_name" width="20%" resize sort align="right">Last Name</mm-grid-column> <mm-grid-column field="email" width="60%" sort>Email</mm-grid-column> </mm-grid> <script> var grid = document.querySelector("#mmTestGrid"), data = []; for(var i=0; i<50; i++){ data.push({ first_name: "First", last_name: "Last", email: "text@example.com" }); } window.addEventListener("WebComponentsReady", function() { grid.data = data; }); </script>
