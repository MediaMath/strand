Attributes

Attribute Options
align : Boolean Default value: horizontal Specifies the direction in which the group should be laid out (horizontal or vertical). 'horizontal''vertical'
fit : Boolean (Optional) Default value: null Instructs the component to fit to its parent container's width (percent or pixel width). 'true''false'

Public Methods

Method Arguements
bindModel(modelproperty) Bind a property on the given model to the input's value. model : Object Default value: undefined The model object to update with value changes. If the model has a '.set()' method, .set(property, inputValue) will be used (otherwise the property field of the model will be assigned directly). property : String Default value: undefined The property on the model that should be bound to the input's value.

Public Properties

Properties Options
value : String Default value: '' the current selected radio value or contents as a string none

Events

Event Event Detail
Type : 'changed' Applies to mm-radio group only checked : Boolean The status of this radio button's selection item : Object A reference to the item which triggered the event
Type : 'selected' Applies to mm-button group only active : Boolean The item's active state index : int The index of the active item item : Object A reference to the item which triggered the event selected : Boolean The item's selected state value : String Value of the selected item, usually it's label
