mm-checkbox

A fully-styled checkbox.

Example HTML Editor

<mm-checkbox></mm-checkbox> <mm-checkbox> <label>Checkbox With Label</label> </mm-checkbox> <mm-checkbox state="partial"></mm-checkbox> <mm-checkbox checked="true"></mm-checkbox> <mm-checkbox checked="true" disabled="true"></mm-checkbox> <mm-checkbox checked="true" icon="check" iconColor="#008000"> <label>Checkbox With a Custom Icon</label> </mm-checkbox>

Example Output

Attributes

Attribute Options
state : String (Optional) Default value: none The display state of the checkbox. 'unchecked''partial''checked'
checked : Boolean (Optional) Default value: false Checked attribute. 'true''false'
disabled : Boolean (Optional) Default value: false Disable the item 'true''false'
icon : String (Optional) Default value: null Turns the checkbox into a custom icon, using icons found in mm-icon. '*see mm-icon for options'
iconColor : String (Optional) Default value: null The hex value to color the icon, passed thru to the underlying mm-icon none

Public Methods

Method Arguements
bindModel(modelproperty) Bind a property on the given model to the input's value. model : Object Default value: undefined The model object to update with value changes. If the model has a '.set()' method, .set(property, inputValue) will be used (otherwise the property field of the model will be assigned directly). property : String Default value: undefined The property on the model that should be bound to the input's value.

Public Properties

Properties Options
checked : Boolean Default value: false Whether or not the checkbox is checked. none
state : String Default value: 'unchecked' String representation of the the checkboxes .checked state ('unchecked', 'checked', or 'partial'). none
icon : String Default value: null Turns the checkbox into a custom icon, using icons found in mm-icon. *see mm-icon for options
iconColor : String Default value: null The hex value to color the icon, passed thru to the underlying mm-icon none

Events

Event Event Detail
Type : 'changed' state : String The new .checked state of the checkbox ('unchecked', 'checked', or 'partial').
