mm-model

Simple model component for 2 way binding to model data

Example HTML Editor

<script> window.addEventListener("WebComponentsReady", function() { document.querySelector("#model").data = {name:'test',link:'http://strand.mediamath.com'}; }); </script> <template is="dom-bind"> <mm-model id="model" data="{{data}}" auto="false"> </mm-model> <mm-input value="{{data.name}}"></mm-input> <mm-input value="{{data.link}}"></mm-input> <mm-action href="{{data.link}}"><label>{{data.name}}</label></mm-action> </template>
