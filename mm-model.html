mm-model

Simple model component for 2 way binding to model data

Example HTML Editor

<template is="auto-binding"> <mm-model id="model"> </mm-model> <mm-input value="{{$.model.data.name}}"></mm-input> <mm-input value="{{$.model.data.link}}"></mm-input> <mm-action href="{{$.model.data.link}}"><label>{{$.model.data.name}}</label></mm-action> </template>

Example Output

Attributes

Attribute Options
mid : String (Optional) Default value: '' Model id. This is mapped to data.id when it exists. none
cid : Collection id. This is assigned to models when they are used in a collection and may be used to reference them temporarily if they have not recieved a server side id. (Optional) none
auto : Boolean/String (Optional) Default value: false Setting for the adapter to automatically sync data to and from server when params or data change 'false''true''load''save'
collection : Object (Optional) Default value: null a reference to the parent collection if one is being used none
adapter : string (Optional) Default value: '' A constructor string for the adapter to inflate for this model to persist changes to that API. This is typically of the format MMAdapterAPINAME none
connector : Object (Optional) Default value: {} A read-only reference to the inflated adapter used for persisting server changes none
defaults : Object (Optional) Default value: null An object hash used to pre-fill a model with data for use in repeating model scenarios. Used in conjuction with init() method none
data : Object (Optional) Default value: {} The data this model stores. none

Public Methods

Method Arguements
init() Prefills the model with any data found in defaults object none
save() Alias to the adapter's post() method. Used to persist data to the api. none
update() Alias to the adapter's put() method. used to persist updates to data to apis that support it. none
fetch() Alias to the adapter's get() method. Used to retrieve data from apis. none
destroy() Alias to the adapter's delete() method. Used to destroy a model on apis that support it. none
clear() Wipes this model's data locally none
get(propertyobj) get a specific attribute from this model property : String Default value: null The property to retrieve from the data object obj : Object (Optional) Default value: this.data Optional functional argument useful for safely obtaining a.b.c data without null errors in scenarios where it is not part of a model. Defaults to instance.data
set(propertyvalueobj) helper method to set a specific attribute's value on the data object property : String Default value: null The property to set data for value : Default value: null The value to save to this property obj : Object Default value: this.data Optional functional argument that allows this method to be used for objects other than this model as required.
