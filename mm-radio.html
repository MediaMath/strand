mm-radio

A fully-styled radio button.

Example HTML Editor

<mm-radio group="aces" unresolved> <label>My Label 1</label> </mm-radio> <mm-radio group="aces" unresolved> <label>My Label 2</label> </mm-radio> <mm-radio group="aces" unresolved> <label>My Label 3</label> </mm-radio> <mm-radio group="aces" checked="true" disabled="true" unresolved> <label>My Label 4</label> </mm-radio>

Example Output

Attributes

Attribute Options
group : String (Optional) Default value: none The name of the group to which the radio button belongs. none
checked : Boolean (Optional) Default value: false Checked attribute. 'true''false'
fit : Boolean (Optional) Default value: null Instructs the component to fit to its parent container's width (percent or pixel width). none
disabled : Boolean (Optional) Default value: false Disable the item none

Public Properties

Properties Options
checked : Boolean Default value: false The status of this radio button's selection none
disabled : Boolean Default value: false The current status of the textarea. When true user input will not be accepted and the field is greyed out. none

Events

Event Event Detail
Type : 'selected' checked : Boolean The status of this radio button's selection item : Object A reference to the item which triggered the event
