Property Description

default : true
options : truefalse
attribute : auto
reflected to attribute : no 		When true, changes to dom configuration will result in AJAX calls being made on the parent object.
autoDebounce : Number  
default : 200
options : none
attribute : auto-debounce
reflected to attribute : no 		The time in milliseconds to debounce dom changes against. This prevents racing calls from keyboard manipulation or other user input binding scenarios.
