mm-modal

A modal container and semi-transparent blocker view to display developer defined content.

Example HTML Editor

<!-- modal markup --> <mm-modal id="testModal"> <div class="content-top"> <mm-header size="medium">Header Content Here</mm-header> <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Etiam convallis viverra sem a ullamcorper. Proin et felis lobortis, vulputate orci eu, posuere ante. Vestibulum interdum augue nec tortor venenatis condimentum. Proin eu erat risus.</p> </div> <!-- mm-footer added for demo only --> <mm-footer fitparent> <mm-action underline> <label>Close</label> </mm-action> <mm-button type="secondary"> <label>Save &amp; Close</label> </mm-button> <mm-button> <label>Save &amp; Add Another</label> </mm-button> </mm-footer> </mm-modal> <!-- button to trigger the modal --> <mm-button id="showBtn" unresolved> <label>Show Modal</label> </mm-button> <!-- styles for developer defined modal content (Note: this is not perscriptive, it can be ANY html content) --> <style> mm-modal p { font-size: 12px; line-height: 18px; margin: 0; } mm-modal mm-header { margin-bottom: 20px; } mm-modal .content-top { padding: 15px; } </style> <!-- script to trigger the modal --> <script language="javascript"> window.addEventListener("WebComponentsReady", function() { var sb = document.getElementById("showBtn"), tm = document.getElementById("testModal"); sb.addEventListener("click", showModal); function showModal(e) { tm.show(); } }); </script>
