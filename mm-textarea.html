mm-textarea

A fully-styled textarea.

Example HTML Editor

<mm-textarea rows="10" placeholder="type here please"></mm-textarea> <mm-textarea rows="10" placeholder="place &lt;code&gt; here" fitparent mono></mm-textarea>

Example Output

Attributes

Attribute Options
placeholder : String (Optional) Default value: none Default text which appears in the textarea none
validation : String (Optional) Default value: none Define which type of validation the textarea should perform. 'empty''int''decimal''email''alpha'
mono : Boolean (Optional) Default value: none Use a monospaced font - good for code. 'true''false'
rows : Int (Optional) Default value: 10 Native - Number of vertical rows of text. none
width : Int (Optional) Default value: native default Native - width of the textarea in pixels. none
height : Int (Optional) Default value: default to rows Native - height of the textarea in pixels. none
maxlength : Int (Optional) Default value: none Native - maximum number of characters. none
fit : Boolean (Optional) Default value: null Instructs the component to fit to its parent container's width (percent or pixel width). none
disabled : Boolean (Optional) Default value: false Disable the item none

Public Methods

Method Arguements
validate() Tells the component to validate its current contents against the validator method none
validator() A method called by validate to check the current field contents. Overwrite this method to provide custom validation. Custom method should return true if validation passes. none
bindModel(modelproperty) provides a backbone model or a POJO to the component which will be updated when its value changes model : Object Default value: undefined The instance of the backbone model or javascript object to be updated when the component value changes property : String Default value: undefined The property on the model object to store the updated value on.

Public Properties

Properties Options
value : String Default value: '' the current textarea contents as a string none
placeholder : String Default value: '' the current placeholder value for the textarea when no contents are set none
disabled : Boolean Default value: false the current status of the textarea. When true user input will not be accepted and the field is greyed out. none
error : Boolean Default value: false Triggers the display of invalid visual state. none
