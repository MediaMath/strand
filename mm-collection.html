mm-collection

A simple collection wrapper component useful for 2 way binding to lists of data

Example HTML Editor

<!-- note this example is intended to show how to use a collection with a data adapter, the example adapter does not exist --> <template is="auto-binding"> <mm-input value="{{terms}}"></mm-input> <mm-collection id="searchResults" adapter="MMAdapterGithub" auto> <q>{{terms}}</q> </mm-collection> <template repeat="{{$.searchResults.data}}"> <div>{{name}}</div> </template> </template>

Attributes

Attribute Options
batchSync : Boolean (Optional) Default value: false Manages adapter batch mode. When true and POSTing a collection, the adapter will make individual model calls instead of trying to post the data in a single call 'true''false'
adapter : string (Optional) Default value: '' A constructor string for the adapter to inflate for this model to persist changes to that API. This is typically of the format MMAdapterAPINAME none
dirtyFetch : Boolean (Optional) Default value: false Indicates that the sorting has changed for the collection so the next fetch should replace all existing data with clean server data. Generally only used by adapters 'true''false'
isLoading : Boolean (Optional) Default value: false Bindable flag to determine if the collection is loading data or not. This is useful for binding a loading indicator's display to. 'true''false'
auto : Boolean/String (Optional) Default value: false Setting for the adapter to automatically sync data to and from server when params or data change 'false''true''load''save'
sparse : Boolean (Optional) Determines if the collection.data should retain empty places for pages that have not yet been fetched. This is useful for showing accurate scrollbars in bound grids. 'true''false'
data : Array (Optional) Default value: [] The array that this collection stores its data in none
page : Number (Optional) The current page number of records to retrieve none
pageSize : Number (Optional) Default value: 10 How many records to retrieve for each call. none
index : Number (Optional) Default value: 0 Index is the raw record that the collection is currently 'pointed' at. This is essentially the page * pageSize with some readahead values none
indexOffset : Number (Optional) Default value: 0 The readahead value used to calculate index. This is useful for fast scrolling lists like grids where the data should be retrieved ahead of users's actions where possible none

Public Methods

Method Arguements
create(initsilent) Creates a new MM-Model instance and appends it to the collection data init : Object Default value: null The data to insert into the model silent : Boolean (Optional) Default value: false If true the collection will not fire an event when the new model is added.
add(modelsilentforce) Adds an existing MM-Model instance to the collection. model : Object MM-Model reference to add to collection. silent : (Optional) Default value: null If true the collection will not fire an event when the new model is added. force : (Optional) Default value: null If true the collection will skip model verification and append to the data array without checking for duplicates.
get(index) Retreieves a model by niave index index : Number Default value: null The index of the model to retreive
getData(index) Retrieves the data for a model at a niave index index : Number Default value: null The index of the model from which to obtain data
delete(model) Deletes a model from the collection model : Object/Index Default value: null May be either an MM-Model object or a niave index to a model in the collection that you wish to delete. Client side deletion only.
clear(silent) Empties the collection. Client side. silent : Boolean Default value: null If true a changed event is not dispatched from the collection when it is emptied
empty(silent) Empties the collection. Client side. silent : Boolean Default value: null If true a changed event is not dispatched from the collection when it is emptied
save() An alias method to post() in the adapter none
update() An alias method to put() in the adapter none
fetch() An alias method to get() in the adapter none
destroy() An alias method to delete() in the adapter none
each(callback) ala forEach in an array, calls the callback for each model in the array and passes it the model callback : Function Default value: null The callback to call for each model. This behaves identically to instance.data.forEach
where(objmatchValues) Return a filtered set of collections based on a matching object. If the 2nd param is true the values in this object must also match. obj : Object Default value: null The matching Object to filter the collection by matchValues : Boolean Default value: null When matchValues is false, the filter will return any models that have the same keys as the matching object. When true the values will be considered as well when filtering.
invalidate() Invalidates the collection when sorting has changed. This assumes that all current data is now invalid and should be replaced with the next fetch. none
