mm-collection

A simple collection wrapper component useful for 2 way binding to lists of data

Example HTML Editor

<template is="dom-bind"> <mm-collection id="searchResults" data="{{dataList}}" auto="false"> </mm-collection> <template is="dom-repeat" items="{{datalist}}"> <div>{{item.name}}</div> </template> </template>
