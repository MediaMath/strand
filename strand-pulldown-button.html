strand-pulldown-button

A fully-styled pulldown button, with multiple options.

Example HTML Editor

<strand-pulldown-button type="primary" unresolved> <strand-list-item>single 3PAS</strand-list-item> <strand-list-item>bulk 3PAS</strand-list-item> <strand-list-item>Upload Assets</strand-list-item> <strand-icon type="plus" width="12" height="12" shapeRendering="auto" optimize="true"></strand-icon> <label>Add Creative</label> </strand-pulldown-button> <strand-pulldown-button type="secondary" unresolved> <strand-list-item>single 3PAS</strand-list-item> <strand-list-item>bulk 3PAS</strand-list-item> <strand-list-item>Upload Assets</strand-list-item> <label>Add Creative</label> </strand-pulldown-button>

Example Output

single 3PAS bulk 3PAS Upload Assets single 3PAS bulk 3PAS Upload Assets
