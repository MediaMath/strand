mm-loader

A fully styled loading graphic with the option for variable messaging. You can override the default spinner graphic by providing your own mm-spinner element.

Example HTML Editor

<!-- Default Loader --> <!-- Note: by default, loaders are positioned absolutely --> <!-- Note: loaders are nested in a div for demonstration purposes only --> <div class="loader-holder" style="width:100%; height:28px;"> <mm-loader id="l1" unresolved> <label>Loading...</label> </mm-loader> </div> <!-- Customized Variant --> <div class="loader-holder" style="width:100%; height:34px;"> <mm-loader id="l2" unresolved bgColor="#003C71" bgOpacity="1"> <!-- You can override the default spinner graphic by providing your own mm-spinner element --> <mm-spinner radius="16" innerRadius="0.8" lineWidth="0.2"></mm-spinner> <label>Waiting...</label> </mm-loader> </div> <!-- Customized Variant --> <div class="loader-holder" style="width:100%; height:84px;"> <mm-loader id="l3" unresolved paddingTop="10" paddingLeft="10" bgColor="#cccccc" bgOpacity="0.75"> <!-- You can override the default spinner graphic by providing your own mm-spinner element --> <mm-spinner radius="32" lineWidth="10" lineWeight="4" fillColor="#003C71"></mm-spinner> </mm-loader> </div>

Example Output

Attributes

Attribute Options
bgColor : String (Optional) Default value: #000000 The color of the loader graphic background. none
bgOpacity : String (Optional) Default value: 0.5 The opacity of the loader graphic background. none
paddingTop : String (Optional) Default value: 5 The paddng, in pixels, applied to both top and bottom of the loader graphic. none
paddingLeft : String (Optional) Default value: 20 The paddng, in pixels, applied to both left and right of the loader graphic. none

Public Methods

Method Arguements
show() Instructs the component to show itself and begin animating. none
hide() Instructs the component to hide itself and stop animating. none
