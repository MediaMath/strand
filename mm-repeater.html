mm-repeater

A component for duplicating a set of form fields from a template

This component contains experimental features. The configuration and API are subject to change. Please use at your own risk.

Example HTML Editor

<mm-repeater id="repeater"> <template preserve-content> <mm-input name="name"></mm-input> <mm-dropdown name="firstOption"> <mm-list-item>Test Item 1</mm-list-item> <mm-list-item>Test Item 2</mm-list-item> <mm-list-item>Test Item 3</mm-list-item> <mm-list-item>Test Item 4</mm-list-item> </mm-dropdown> <mm-group name="secondOption"> <mm-radio><label>Test 1</label></mm-radio> <mm-radio><label>Test 2</label></mm-radio> <mm-radio><label>Test 3</label></mm-radio> </mm-group> </template> </mm-repeater>
